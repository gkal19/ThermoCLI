## Instalar

O projeto usa `Node` e `npm`, então você pode instalar globalmente usando o comando:
```shell
$ npm i -g thermocli
```

## Uso

Para rodar o aplicativo é só acessar qualquer terminal usando o comando:
```shell
$ thermo
```
